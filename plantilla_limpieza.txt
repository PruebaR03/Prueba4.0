# CONFIGURACIÓN DE LIMPIEZA POST-ENRIQUECIMIENTO
# Elimina filas según grupos de condiciones

# SINTAXIS:
# Eliminar si: columna = valor &&  → AND (conecta con la siguiente condición)
# Eliminar si: columna = valor ||  → OR (alternativa, inicia nuevo grupo)
# Eliminar si: columna = valor     → Sin operador (última del grupo)

# OPERADORES DE COMPARACIÓN:
# columna = valor          → Igual exacto
# columna = +valor+        → Contiene "valor"
# columna = *+valor+*      → NO contiene "valor"
# columna = *valor*        → Diferente de "valor"
# columna = []             → Vacío (null, vacío, N/A)
# columna = *[]*           → NO vacío

# LÓGICA:
# - Condiciones con && se agrupan (todas deben cumplirse)
# - Grupos se conectan con OR implícito entre ellos
# - Ejemplo: (A && B && C) || (D && E) → Elimina si cumple (A Y B Y C) Ó (D Y E)
